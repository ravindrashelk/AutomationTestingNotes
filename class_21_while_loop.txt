while loops:-
-----------

if we want to execute a group of statements itertivly until some condtion false then we should go for while loop.

syntax:

while condition:
	body/statements


1.WAP to print 1 to 10 using while loop?

var = 1
while var<= 10:
    print(var)
    var = var + 1

2. WAP to add 1 to 10?

var = 1
sum = 0
while var<= 10:
    sum = sum + var
    var = var + 1
print(sum)
